### Quick Setup (5 minutes)

```bash
# 1. Install Fly CLI
brew install flyctl

# 2. Login
fly auth login

# 3. Launch (creates Dockerfile + config automatically)
fly launch --name stealth-scanner

# 4. Set secrets
fly secrets set TELEGRAM_BOT_TOKEN=your_token
fly secrets set TELEGRAM_CHAT_ID=your_chat_id
# Add API keys as needed (OPENAI_API_KEY, etc.)

# 5. Deploy
fly deploy
```

public URL like `https://stealth-scanner.fly.dev` and bot will work immediately.

---

### One Caveat

The first time deploy, it asks for credit card (for identity verification), but **you won't be charged**. You can skip this with:
```bash
fly launch --name your-app --no-deploy
# Then manually set card to "Free" in billing settings
```
